📊 Cómo conectar Google Sheets

1. Abre Google Sheets y crea una hoja nueva
2. Ve a Extensiones → Apps Script
3. Borra el código que hay y pega el de abajo
4. Haz clic en Implementar → Nueva implementación
5. Tipo: App web · Acceso: Cualquiera
6. Copia la URL y pégala arriba

Código para Apps Script:

function doPost(e) {
  const s = SpreadsheetApp
    .getActiveSheet();
  const d = JSON.parse(
    e.postData.contents);
  s.appendRow([d.name, d.date,
    d.time,
    d.late?'Tarde':'A tiempo',
    d.geo]);
  return ContentService
    .createTextOutput('ok');
}

📱 Cómo crear bot de Telegram

1. Abre Telegram y busca @BotFather
2. Escribe /newbot y sigue los pasos
3. Copia el token que te da
4. Para el Chat ID busca @userinfobot
5. Si quieres un grupo, añade el bot y usa @RawDataBot para ver el ID del grupo
